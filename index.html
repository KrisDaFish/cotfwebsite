What You Will Do In This Course

01

Learn the fundamentals and basics of programming

This includes the logic behind programming, and the thought-processes. This part of the course will involve interactive activities so students thoroughly understand the concepts, while also having fun.

02

Learn how to start coding Python

Learn how to use the logic behind coding in a practical setting. This includes learning how to convey logical operations through flowcharts and diagrams, and then later implementing these operations through python. Using this knowledge, they will learn how to solve basic problems using code/python.

03

Make your first project in Python

After learning and developing basic problem-solving skills, use these skills to make your first coding project in Python!

04

Absolute Beginners Welcome!

This course is made for absolute beginners in coding and programming! It requires zero prior knowledge or experience, so don't feel intimidated!
